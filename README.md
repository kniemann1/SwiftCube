# SwiftCube
A beginning Ipad app that displays a rotating photo cube. 

Really it is an experiment on how to integrate AI coding into an Ipad Swift Playgrounds environment. 
Because Swift Playgorunds is "sandboxed" running Cluade or Codex directly on the Ipad will have no benefit. 

Claude suggests using Github as an intermediary. 

# The daily loop

Code in Swift Playgrounds → switch to a-Shell → lg2 add . && lg2 commit -m "..." && lg2 push
Changes on GitHub → fully quit Swift Playgrounds (swipe it away) → lg2 pull → reopen. Never pull while Playgrounds is running.
